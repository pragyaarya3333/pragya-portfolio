# Pragya Arya Portfolio

A responsive frontend portfolio website customized for my resume and personal projects.

## About

This project is a customized portfolio website built using:

- Lightning Web Components OSS
- JavaScript
- HTML
- CSS
- Salesforce Lightning Design System

It includes sections for:

- About
- Skills
- Experience
- Projects
- Education

## Features

- Responsive portfolio layout
- Light and dark theme support
- Skills section
- Experience section
- Projects section
- Education section
- GitHub integration
- Custom profile, footer, and browser tab title
